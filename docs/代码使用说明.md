# 代码使用说明

## 一、用途

本代码包用于复现论文《福州市商品房价格的影响因素分析与房价预测》中的主要实证结果，包括：

1. 数据读取与字段校验；
2. 描述性统计；
3. 影响房价因素的可视化分析；
4. 主成分分析（PCA）；
5. 基于 PCA 的多元线性回归模型（PCA-MLR）；
6. 基于 PCA 的人工神经网络模型（PCA-ANN）；
7. 全样本拟合比较；
8. 以 2001–2019 为训练集、2020–2021 为检验集的类预测比较；
9. 论文图表批量导出。

## 二、如何运行

### 方式 1：命令行一键运行

在代码包根目录执行：

```bash
python run_all.py
```

若你使用 Windows，也可直接双击 `run_all.bat`。

### 方式 2：Jupyter Notebook 运行

打开：

```text
notebooks/reproduce_all_results.ipynb
```

按顺序运行全部单元即可。

## 三、输出位置

程序运行结束后，所有结果会输出到：

```text
outputs/
├─ figures/
└─ tables/
```

其中：
- `figures/` 保存论文用图；
- `tables/` 保存论文用表及扩展统计表；
- `tables/thesis_tables.xlsx` 为表格总汇；
- `tables/summary_results.json` 为关键结果摘要。

## 四、脚本说明

### 1. `src/config.py`
集中定义：
- 原始变量名；
- 英文简称；
- 图表题注；
- 理论预期；
- 模型参数。

### 2. `src/data_utils.py`
负责：
- 读取 Excel；
- 校验列名；
- 返回建模所需数据框。

### 3. `src/descriptive_analysis.py`
负责生成：
- Table 2-1、Table 2-2；
- Figure 1-1、Figure 2-1 ～ Figure 2-6；
- VIF 与相关系数附加表。

### 4. `src/pca_analysis.py`
负责生成：
- Table 3-1、Table 3-2；
- Figure 3-1、Figure 3-2。

### 5. `src/modeling.py`
负责：
- PCA-MLR 与 PCA-ANN 模型训练；
- 训练集与检验集评价；
- Table 5-1 ～ Table 5-4；
- Figure 5-1、Figure 5-2。

### 6. `src/reporting.py`
负责：
- CSV / Excel / JSON 结果导出；
- 图表清单生成。

## 五、关键建模设定

- 主成分提取方法：标准化后 PCA；
- 主成分保留数：3；
- 神经网络模型：`MLPRegressor(hidden_layer_sizes=(3,), activation='tanh', solver='lbfgs', alpha=0.01, max_iter=2000, random_state=42)`；
- 训练集：2001–2019；
- 检验集：2020–2021；
- 全样本拟合用于比较两类模型的样本内拟合能力。

## 六、如何更换数据

若要改成其他城市，只需：

1. 替换 `data/data.xlsx`；
2. 保持列名与当前 Excel 一致；
3. 重新运行 `python run_all.py`。

## 七、标注说明

为了便于后续修改，本代码已做以下标注：

- 每个脚本顶部均有用途说明；
- 关键函数带有 docstring；
- 关键步骤（标准化、PCA、训练/测试划分、指标计算、绘图）均写有中文注释；
- 输出文件名与论文图表编号一一对应，便于直接查找和核对。
